Recursive descent parsers zijn parsers die geïmplementeerd zijn als een stel recursieve functies. Per grammaticale regel is er één functie. Het "descent"-gedeelte van de naam komt van het feit dat we beginnen in de wortel en dan steeds dieper in de structuur gaan.

Recursive descent parsers zijn een uiterst belangrijke groep parsers, maar niet de enige groep. Het is wel de enige groep die we met ANTLR kunnen maken.

Een voorbeeld van toekenning: // assign : ID '=' expr ';' ; void assign() { // method generated from rule assign match(ID); // compare ID to current input symbol then consume match('='); expr(); // match an expression by calling expr() match(';'); }

Een toekenning kan maar op één manier geschreven worden. Een grammaticale regel kan ook meerdere alternatieven omvatten.
